Se deberá de realizar una práctica individual muy sencilla para familiarizarte con Git donde se manejará el repositorio local:



Crea una carpeta en tu equipo con el nombre MiPrimerProyectoGit.



Abre esa carpeta desde Git Bash (o terminal) y analiza el comando git init para iniciar un repositorio.



Crea dentro de la carpeta un archivo llamado readme.txt con una breve descripción de lo que aprenderás con Git.



Guarda los cambios y analiza los siguientes comandos en la terminal:


git add 


git commit -m "Primer commit de prueba"





Cuando termines, toma una captura de pantalla mostrando los comandos utilizados y el mensaje de confirmación del commit.
